정의(Definition)

  • <circle> 요소는 SVG 기본 모양으로 중심점과 반지름을 기준으로 원을 그리는데 사용됩니다.

.svgBox .svg {width: 160px; height: 160px; background: #ffebee;}
.circle1 {
    fill: #f48fb1; cx: 80; cy: 80; r: 60;
}
.circle2 {
    fill: #f48fb1; cx: 80; cy: 80; r: 60;
    stroke: #880e4f;
    stroke-width: 5;
}
